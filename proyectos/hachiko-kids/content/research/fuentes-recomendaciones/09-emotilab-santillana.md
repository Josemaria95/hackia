# Emotilab (Santillana) — Fuente para Motor de Recomendaciones

## Clasificación para el motor

| Criterio | Evaluación |
|----------|-----------|
| **Utilidad para recomendaciones** | **MEDIA** (como referencia de framework) |
| **Respaldo institucional** | Alto — Santillana (editorial educativa líder) + marco CASEL |
| **Datos estructurados** | Sí — sesiones estructuradas por competencia CASEL y nivel escolar |
| **Determinístico** | Parcial — sesiones prediseñadas pero no protocolos IF-THEN |
| **Evidencia base** | CASEL (Collaborative for Academic, Social, and Emotional Learning) — Daniel Goleman |

---

## Qué contenido sirve para recomendaciones

### 1. Framework CASEL como organizador de dimensiones

CASEL define 5 competencias que mapean bien a las dimensiones de Hachiko:

| Competencia CASEL (Emotilab) | Dimensión Hachiko | Utilidad |
|---|---|---|
| Autoconciencia | Ánimo general | Framework para clasificar recomendaciones |
| Autorregulación | Regulación emocional | Misma dimensión — contenido validado |
| Conciencia social | Socialización | Marco para interpretar patrones sociales |
| Habilidades de relación | Conducta prosocial | Marco para tips de interacción |
| Toma de decisiones responsable | Seguimiento instrucciones | Marco para tips de autonomía |

### 2. Sesiones por nivel escolar

Emotilab tiene sesiones de 1° básico a III medio. La estructura nivel→sesión→competencia podría informar la segmentación por edad de las recomendaciones.

---

## Reglas determinísticas extraíbles

No se pueden extraer reglas directas — Emotilab es un **competidor** con contenido propietario. Lo útil es el **framework CASEL** (que es público):

```
FRAMEWORK CASEL (público, no propiedad de Emotilab):
- IF baja autoconciencia → recomendaciones de vocabulario emocional
- IF baja autorregulación → técnicas de pausa y respiración
- IF baja conciencia social → ejercicios de perspectiva ("¿cómo se siente el otro?")
- IF bajas habilidades de relación → juego cooperativo
- IF baja toma de decisiones → ofrecer opciones limitadas
```

---

## Limitaciones

- **Es competidor directo** — su contenido es propietario y no puede usarse como fuente.
- Opera en el mismo canal (colegios) — usar su contenido sería problemático comercialmente.
- El framework CASEL es público y se puede referenciar directamente sin pasar por Emotilab.
- Sin presencia digital (no tiene app) — no hay datos abiertos.

## Veredicto

**NO sirve como fuente de recomendaciones.** Emotilab es competidor. Lo que sirve es el **framework CASEL** (público), que se puede referenciar directamente. Hachiko Kids debería diferenciarse de Emotilab ofreciendo lo que ellos no tienen: app gamificada + datos continuos + canal directo a familias (no solo docentes).
